# 通用 Web 搜索工具

用一个搜索 API 替代分别维护小红书/两步路/大众点评等各自的爬虫。

## 背景

小红书、两步路调研后确认无合规 API，分别维护爬虫成本高、风险大。一个通用搜索工具可以一次覆盖所有外部数据需求，让 DeepSeek 按需搜索任意网站。

## 候选方案

| 方案 | 免费额度 | 国内可用 | 备注 |
|---|---|---|---|
| **Brave Search API** | 2000 次/月免费 | 需代理 | 独立索引，不依赖 Google |
| **必应搜索 API (Azure)** | 1000 次/月免费 | 需代理（Azure） | 结果质量好，有中文支持 |
| **Serper API** | 2500 次免费试用 | 需代理 | Google 结果 |
| **百度搜索** | 无公开 API | ✅ | 只有爬虫方案，不合规 |
| **夸克/360搜索** | 无公开 API | ✅ | 同上 |

## 待办

- [ ] 确认选哪个（倾向 Brave，独立索引 + 便宜）
- [ ] 加 `web_search(query, site?)` tool：`site` 可选，如 `site:xiaohongshu.com`，让 DeepSeek 决定要不要缩范围
- [ ] 结果截断：只返回 top 5 摘要，避免 token 爆炸
- [ ] 国内需要代理，配 `SEARCH_PROXY` 或复用 `HTTPS_PROXY`

## 注意点

- 搜索结果仍需告知用户"数据来自公开网络，可能不准确"。
- 不要把搜索结果直接存库（版权问题，只用于当次回答）。
