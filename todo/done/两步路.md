# 两步路

查询数据来源接入两步路（户外轨迹）。

## 调研结论（2026-06-24）

**结论：当前不接入，记录备用。**

### API 现状

两步路没有任何公开开发者 API。平台是封闭的移动 app（iOS/Android），轨迹数据是核心商业资产（付费会员解锁）。翻遍官网和 GitHub 没有找到任何开放接口文档。

### 技术可行性

可以逆向 app 的 HTTPS 接口，但：
- 接口无文档，随时可能变更
- 需要维护登录态（有设备绑定）
- 违反用户协议
- 轨迹数据有版权（用户上传，版权归用户/平台共有）

### 实际使用场景评估

用户在群里聊"去四明山爬山"，想要轨迹推荐。这个场景目前可以通过：
1. DeepSeek 从训练数据里知道热门路线的基本信息
2. Bot 主动提示"可以去两步路 App 搜「XX」"

效果接近，但不需要维护一个不稳定的爬虫。

### 替代路径

如果未来要做户外活动的数据支撑，更可持续的方式是接入一个**通用 Web 搜索 API**，让 DeepSeek 搜索两步路站内内容（通过 `site:2bulu.com` 等方式）——比维护私有爬虫可靠。
