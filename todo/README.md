## 编辑约定

- 新增 todo：在对应主题目录下加 `<短名>.md`，文件内按「背景 / 现状 / 待办 / 注意点」写；只有一两句也可以一条 list 搞定。
- 新增主题：建子目录，前缀沿用 `0/1/2-` 表示优先级，同时在本文件加链接。
- 完成后：把文件 `git mv` 到 `done/`，并在文件末尾加 `## 完成记录`（日期 / commit / 备注）。
- 文件名用中文 / 短横线，避免空格。
